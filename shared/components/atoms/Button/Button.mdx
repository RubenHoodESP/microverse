# Button Component

El componente Button es un componente atómico que proporciona una interfaz de usuario consistente para acciones y formularios.

## Características

- Múltiples variantes (default, destructive, outline, secondary, ghost, link)
- Diferentes tamaños (default, sm, lg, icon)
- Soporte para estado disabled
- Totalmente accesible
- Personalizable con Tailwind CSS

## Uso

```tsx
import { Button } from '@/shared/components/atoms/Button';

export default function MyComponent() {
  return (
    <Button variant="default" size="lg">
      Click me
    </Button>
  );
}
```
